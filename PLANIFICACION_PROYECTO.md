# Planificación del Proyecto Papelería - BLOC-2

## 1. Casos de Uso
El sistema facilita la interacción entre el empleado y las operaciones centrales del negocio.

### Caso de Uso 1: Registro de Venta
*   **Descripción:** El vendedor registra productos y procesa el pago. El inventario se actualiza en tiempo real.

### Caso de Uso 2: Alertas de Inventario
*   **Descripción:** El sistema notifica al administrador cuando los productos alcanzan su nivel mínimo.
